1.3.3 python开发对应

装有anaconda系列虚拟环境即可，在这里先创建一个用于后面项目使用的虚拟环境，centos已提供miniconda2环境(单机版&分布式版本都已经安装了所有Python环境)

conda create -n reco_sys python=3.6.7

pip install -r requirements.txt --ignore-installed
